Linux常用命令

发表于 2015-07-30   |   分类于 Linux   |     |  
  • awk:强大复杂的文本分析工具(见附注)
  • bzip2:bz2文件压缩解压工具
    • -d:解压
  • cat:查看文件
    • cat /dev/null > xxx:清空文件内容
  • cd:转移路径
  • cp:复制文件(cp patha pathb)
    • -f:如果目标文件无法打开则将其移除并重试(无提示)
    • -R/-r:递归复制目录及其子目录
  • df:查看文件系统的大小
    • -h:以容易理解的格式列出文件大小
    • -l:列出文件的详细信息
  • du:显示文件或目录的磁盘占用量,格式为du [option] 目录或文件(默认为当前目录) [附加命令]
    • –max-depth=N(如果N为0,则统计当前目录总大小;如果N为1,则统计当前目录及其直接子目录的大小;依次类推)
    • -h:以容易理解的格式列出文件大小
    • -l:列出文件的详细信息
  • find:格式为find [OPTION] path [expression]
    • find path -name xxx:在path下查找匹配xxx的文件(默认当前目录)
    • -name:按名字查找
    • -type:按类型查找
      • -d:文件类型为目录
  • grep:文本搜索工具,如果搜索来源为文件则源置后,否则源置前
    • -P:使用正则表达式
    • -i:忽略大小写
  • head:显示前几行
    • -n:显示前n行
  • iptables:防火墙相关
    • –dport:目的端口
    • –sport:源端口
    • -A:追加新规则(INPUT/OUTPUT)
    • -L:显示所有规则
    • -d:目的地址
    • -i:输入接口
    • -j:执行目标(ACCEPT/DROP/QUEUE/RETURN)
    • -o:输出接口
    • -p:指定协议
    • -s:源地址
  • jar
    • -c:创建新jar包
    • -f:指定jar文件名
    • -M:不生成清单文件,此命令会忽略-m参数
    • -m:指定manifest清单文件
    • -u:更新存在的jar文件
    • -v:生成详细报告并打印到标准输出
    • -x:展开jar文件包中的指定文件或所有文件
  • last:查看用户登录、注销和重启终端的记录
  • ln:格式为ln [option] 原文件 要生成的链接
    • -s:创建符号链接而非硬链接
  • ls:显示当前路径下的文件和目录
    • -A:列出除.及..以外的任何项目
    • -a:列出目录下所有文件,包括以.开头的隐含文件
    • -h:以容易理解的格式列出文件大小
    • -k:以K为单位列出文件大小
    • -l:列出文件的详细信息(部分系统可以简写为ll)
  • lsb_release:查看当前发行版
    • -a:查看发行版版本的所有信息
  • make:编译
    • make install &> xxx:将安装信息存入日志文件
    • make install:安装
  • mv:移动文件及单文件重命名(mv patha/filea pathb/fileb:如果filea与fileb文件名不同,则重命名 )
    • -f:覆盖前不询问
    • -v:显示详细信息
  • netstat:查看端口状态
    • -a:查看全部状态的端口(监听和非监听)
    • -e:查看所有附加信息
    • -l:只显示监听状态的端口
    • -n:使用数字而不是主机名/域名显示连接的主机(IP + 端口)
    • -p:查看使用当前端口的程序的PID和名称
    • -t:仅显示tcp相关
  • pwd:显示当前路径
  • rm:
    • -f:删除前不询问
    • -r:递归删除目录及其子目录
  • sort:按字母排序(注意:被分隔的重复行不计入重复行)
    • -k:指定间隔符(默认为空格)
    • -n:按数字排序
    • -r:反序
    • -t:指定按第几列排序
  • tail:查看文件最后10行
    • -c:指定输出字节数
    • -f:随文件变化而变化
    • -n:指定输出行数(可省略n,eg:tail -10 = tail -n 10)
    • –pid:指定进程号，在进程死掉后输出结束
    • -q:不显示处理详情
    • -s:指定休眠间隔(单位为秒)
    • -v:显示详情
  • tar
    • -c:创建新压缩文件
    • -f:使用归档文件,后须直接连接待解压文件
    • -v:详细列出处理文件
    • -x:解压文件
    • -z:使用gzip格式(压缩或解压)
  • tr:字符或字符串替换–tr source dst
    • -s:删除多个连续字符或字符串,只保留一个–tr -s ‘ ‘ ‘\n’
  • uname:查看当前系统
    • -a:linux系统的详细信息号
    • -m:查看CPU型号
  • uniq:报告或删除文件中的重复行
    • -c:在输出行前面加上每行在文件中出现的次数
    • -d:只显示重复行
  • unzip
    • -l:查看zip文件内容而不解压
    • -t:验证zip文件完整性
    • -v:查看zip文件详细信息(不解压)
  • wc
    • -c:统计字节数
    • -l:统计行数
    • -w:统计字数
  • zip:格式为zip [option] 压缩文件名 待压缩文件路径(如为多个以空格分隔)
    • -r:递归压缩目录及其内的子目录
  • awk:
  • 简单示例:
    • awk ‘{print $1,$4}’ file.txt
    • 单引号中被大括号包着的即为awk的语句
    • \$0表示整行
    • $n表示第n列
  • 格式化示例:

    awk ‘{printf “%-8s %2s\n”,$2,$6}’ file.txt

    • 格式化输出与C语言相同
    • s表示字符串
    • -8表示从左边算起每列长度一共为8
  • 过滤示例:

    awk ‘$3==0 && $5==”tcp”‘ file.txt

    • 可以使用的比较运算符有==、>=、<=、>、<、!=
  • 指定分隔符示例:

    awk -F OFS=”\t” ‘{print $1,$3,$6}’ file.txt

  • 模式匹配示例:

    awk ‘/LISTEN/‘ file.txt
    awk ‘$6 ~ /FIN|TIME/ || NR==1 {print NR,$4,$5,$6}’ OFS=”\t” file.txt

    • ‘/ /‘内即为需要匹配的字符串
    • ~为匹配,!~为不匹配
  • 内建变量
    • FS:输入字段分隔符,默认为空格或Tab
    • NF:列数
    • NR:行数
    • RS:输入记录分隔符,默认为换行符
    • OFS:输出字段分隔符,默认为空格或Tab
    • ORS:输出记录分隔符,默认为换行符
    • FILENAME:当前输入文件的名字
  • 参数
    • -F:指定域分隔符,如果不指定,默认为空格
