Idea快捷键

发表于 2015-03-14   |   分类于 Tool   |     |  

快捷键:

  • Alt + F12:打开命令行
  • Alt + Shift + Up/Down:将当前行上下移动
  • Ctrl + /:注释选中行(//)
  • Ctrl + Alt + Enter:向上插入一行
  • Ctrl + Alt + L:代码格式化
  • Ctrl + Alt + O:移除不需要的import
  • Ctrl + Alt + S:进入设置
  • Ctrl + Alt + Shift + S:进入项目设置
  • Ctrl + B:查看声明
  • Ctrl + D:复制当前行到下一行
  • Ctrl + E:显示最近文件列表
  • Ctrl + F:本文件内查询
  • Ctrl + F4:关闭当前TAB
  • Ctrl + F8:添加/移除断点
  • Ctrl + F12:文档结构
  • Ctrl + K:提交代码到VCS
  • Ctrl + N:在当前打开的TAB中查找文件
  • Ctrl + Shift + /:注释选中行(/**/)
  • Ctrl + Shift + A:查找IDEA命令、菜单、操作
  • Ctrl + Shift + Enter:向下插入一行
  • Ctrl + Shift + F10:运行当前文件的主函数
  • Ctrl + Shift + F12:最大化编辑窗口
  • Ctrl + Shift + N:查找文件(包括引入的jar包中的文件)
  • Ctrl + W:逐层选择编辑器中的文字
  • Ctrl + Y:删除当前行
  • F7:单步into
  • F8:单步over
  • F9:恢复执行
  • Shift Double:快速双击Shift－查找recent files/tool window/all files
  • Shift + Enter:向下插入一行
  • Shift + F6:修改命名
  • Shift + F10:运行当前程序

技巧:

  • 设置->Editor->General:Change font size(zoom) with Ctrl+Mouse Wheel选中后,可以在编辑器中使用Ctrl+鼠标滚轮改变字体大小
