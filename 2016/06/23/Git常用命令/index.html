gitignore用法

  • 空行或以#开头的行会被忽略
  • 可以使用标准glob模式匹配（glob模式即shell使用的简化了的正则表达式）
    • *匹配0或多个字符
    • ?匹配一个任意字符
    • []中使用-分隔两个字符表示匹配这两个字符范围内的字符
    • []中多个字符表示匹配多字符中的任意一个字符
  • 匹配模式最后跟/说明要忽略的是目录
  • 要忽略指定模式以外的文件或目录，可以在模式前加!
